¿Que versiones de java script hay en el mercado?

    Javascript 1: javascript 1 nacio con el Netscape 2.0 y es el que soportaba gran cantidad de instrucciones y funciones, casi todas las que existen ahora ya se introdujeron en el primer estándar

    Javascript 1.1: Esta es la versión de javascript que se diseño con la llegada de los navegadores 3.0 implementaba poco mas que su anterior versión.

    Javascript 1.2: Esta versión de los navegadores 4.0 esta tiene como desventaja que es un poco distinta en plataformas Microsoft y Netscape, ya que ambos navegadores crecieron de distinto modo y estaban en plena lucha por el mercado.

    Javascript 1.3: esta versión que implementan los navegadores 5.0. en esta versión se han limado algunas diferencias y asperezas entre los dos navegadores.

    Javascript 1.5: Esta versión actual, en el momento de escribir estas líneas, que implementa Netscape 6.

    Por su parte, Microsoft también ha evolucionado hasta presentar su versión 5.5 de JScript .

    En 2009 se publico ECMAscript 5. Esta versión que vamos a tratar principalmente en el manual, que funciona en todos los navegadores del mercado.

    En 2015 se lanzo la sexta versión de ECMAscript, con una cantidad importante de novedades. En esta versión se puede usar completamente, excepto en internet explorer.

    Actualmente se adquirió un compromiso de liberar una versión del estándar ECMASCRIPT por año. Por ello existen diversos estándares con pequeños cambios incrementales, que se han ido incorporando a distintos ritmos entre los navegadores.
